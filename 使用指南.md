# ParticleGame - 完整使用指南

## 📂 文件夹内容

```
ParticleGame/                     总大小: 108KB
├── particle_game.py              游戏主程序 (64KB)
├── launcher.py                   智能启动器 (4KB)
├── requirements.txt              依赖列表
├── README.md                     详细文档
├── BUILD_INSTRUCTIONS.md         打包教程
├── build_game.sh                 Linux/Mac打包脚本
├── build_game.bat               Windows打包脚本
├── package_release.sh           一键发布脚本
├── 快速开始.txt                  快速指南
└── 文件说明.txt                  文件说明
```

## 🎮 三种玩游戏的方式

### 方式1：使用智能启动器（最简单）

```bash
cd ParticleGame
python launcher.py
```

**优点**：自动检查和安装依赖，一键启动

---

### 方式2：直接运行

```bash
cd ParticleGame
pip install -r requirements.txt  # 首次需要安装依赖
python particle_game.py
```

---

### 方式3：运行可执行文件（需先打包）

```bash
cd ParticleGame
./build_game.sh    # Linux/Mac
# 或
build_game.bat     # Windows

# 然后运行
./dist/ParticleGame
```

---

## 📤 三种分享给朋友的方式

### 方式1：打包整个文件夹（最简单）

```bash
# 在ParticleGame的上级目录
zip -r ParticleGame.zip ParticleGame/
```

**发送给朋友**：ParticleGame.zip
**朋友使用**：解压后运行 `python launcher.py`
**优点**：简单，跨平台通用
**缺点**：朋友需要有Python环境

---

### 方式2：打包成可执行文件（最专业）

```bash
cd ParticleGame
./build_game.sh    # Linux/Mac
# 或
build_game.bat     # Windows
```

**发送给朋友**：dist/ParticleGame 或 dist/ParticleGame.exe
**朋友使用**：直接双击运行
**优点**：朋友无需Python，直接运行
**缺点**：文件较大(200-300MB)，每个平台需单独打包

---

### 方式3：一键打包发布（最全面）

```bash
cd ParticleGame
./package_release.sh
```

这会在 `release/` 目录生成：
- `ParticleGame_v1.0_Source.zip` - 源代码包（给开发者）
- `ParticleGame_v1.0_Linux.tar.gz` - 可执行文件包（给普通用户）

---

## 🚀 推荐的分享流程

### 情况A：朋友有Python环境

1. **你的操作**：
```bash
cd /home/pc
zip -r ParticleGame.zip ParticleGame/
# 将 ParticleGame.zip 发给朋友
```

2. **朋友的操作**：
```bash
unzip ParticleGame.zip
cd ParticleGame
python launcher.py
```

---

### 情况B：朋友完全不懂编程

1. **你的操作**：
```bash
cd ParticleGame
./build_game.sh    # 打包（耗时3-5分钟）
# 将 dist/ParticleGame 发给朋友
```

2. **朋友的操作**：
```bash
# Linux/Mac
chmod +x ParticleGame
./ParticleGame

# Windows
双击 ParticleGame.exe
```

---

### 情况C：分享给所有人（GitHub）

1. **创建GitHub仓库**：
```bash
cd ParticleGame
git init
git add .
git commit -m "Initial release"
git remote add origin <your-repo-url>
git push -u origin main
```

2. **在不同平台打包**：
- 在Linux上打包 → Linux可执行文件
- 在Windows上打包 → Windows可执行文件
- 在Mac上打包 → Mac可执行文件

3. **在GitHub创建Release，上传所有文件**

4. **分享GitHub链接给朋友**
