# 🚀 快速打包Windows EXE - 最简单的方法

## 方法1：GitHub Actions（最推荐 ⭐⭐⭐⭐⭐）

**优点：**
- ✅ 完全自动化，一次配置永久使用
- ✅ 在真实的Windows环境中打包，100%可靠
- ✅ 免费，无需本地配置
- ✅ 可以同时打包多个平台

**步骤：**

### 1. 创建GitHub仓库并上传代码

```bash
cd /home/pc/ParticleGame

# 初始化Git（如果还没有）
git init
git add .
git commit -m "Initial commit"

# 在GitHub上创建新仓库，然后：
git remote add origin https://github.com/你的用户名/ParticleGame.git
git push -u origin main
```

### 2. 创建Release触发打包

在GitHub网页上：
1. 点击 "Releases" → "Create a new release"
2. 输入标签：`v1.0`
3. 点击 "Publish release"

**完成！** GitHub会自动在Windows机器上打包，几分钟后就能下载EXE文件。

## 方法2：使用Windows虚拟机（如果有）

如果你有Windows虚拟机或双系统：

1. 在Windows中安装Python 3.9
2. 运行：
   ```bash
   pip install -r requirements.txt pyinstaller
   pyinstaller --onefile --name ParticleGame particle_game.py
   ```
3. 完成！EXE在 `dist/` 目录

